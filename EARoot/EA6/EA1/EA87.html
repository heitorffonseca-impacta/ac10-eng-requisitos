Scenario Type
CANCELAMENTO DE AGENDAMENTO PACIENTE Basic Path
Details:
Este cenario serve para o paciente cancelar o agendamento.
Steps:
1. O PACIENTE SELECIONA A OPÇÃO DE CANCELAR AGENDAMENTO
2. O SISTEMA EXIBE A TELA DOS AGENDAMENTOS
3. O PACIENTE SELECIONA O AGENDAMENTO QUE SERÁ CANCELADO
4. O SISTEMA EXIBE A TELA PARA INFORMAR O MOTIVO DO CANCELAMENTO
5. O PACIENTE ESCREVE O MOTIVO DO CANCELAMENTO DO AGENDAMENTO E CLICA EM “PROSSEGUIR”
6. O SISTEMA EXIBE A TELA DO CANCELAMENTO DO AGENDAMENTO
7. O PACIENTE CLICA NO BOTÃO DE CANCELAR AGENDAMENTO
8. O SISTEMA REGISTRA O CANCELAMENTO DO AGENDAMENTO E NOTIFICA O PROFISSIONAL
