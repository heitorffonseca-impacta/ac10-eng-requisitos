Scenario Type
Re-agendamento de consulta paciente Basic Path
Details:
Steps:
1. O PACIENTE SELECIONA A OPÇÃO DE REMARCAR CONSULTA
2. O SISTEMA EXIBE A TELA DOS AGENDAMENTOS
3. O PACIENTE SELECIONA O AGENDAMENTO QUE SERÁ REMARCADO
4. O SISTEMA EXIBE AS DATAS E HORÁRIOS DISPONÍVEIS DO PROFISSIONAL
5. O PACIENTE SELECIONA UMA DATA E UM HORÁRIO DISPONÍVEL E CLICA NO BOTÃO “PROSSEGUIR”
6. O SISTEMA EXIBE UMA TELA DE CONFIRMAÇÃO DO RE-AGENDAMENTO
7. O PACIENTE CLICA NO BOTÃO SOLICITAR RE-AGENDAMENTO
8. O SISTEMA REGISTRA A SOLICITAÇÃO DE RE-AGENDAMENTO E NOTIFICA O PROFISSIONAL
Re-agendamento de consulta do profissional Alternate
Details:
Steps:
1. O PROFISSIONAL SELECIONA A OPÇÃO DE REAGENDAR CONSULTA
2. O SISTEMA EXIBE A TELA DOS AGENDAMENTOS
3. O PROFISSIONAL SELECIONA O AGENDAMENTO QUE SERÁ REMARCADO
4. O SISTEMA EXIBE OS DADOS DO CLIENTE QUE TERÁ A REMARCAÇÃO
5. O PROFISSIONAL CONFIRMA OS DADOS DO CLIENTE QUE TERÁ A REMARCAÇÃO
6. O SISTEMA EXIBE AS DATAS E HORÁRIOS DISPONÍVEIS PARA O REAGENDAMENTO
7. O PROFISSIONAL SELECIONA UMA DATA E UM HORÁRIO DISPONÍVEL E CLICA NO BOTÃO “PROSSEGUIR”
8. O SISTEMA EXIBE UMA TELA DE CONFIRMAÇÃO DA REMARCAÇÃO
9. O PROFISSIONAL CLICA NO BOTÃO DE RE-AGENDAMENTO
10. O SISTEMA REGISTRA A SOLICITAÇÃO DE RE-AGENDAMENTO E NOTIFICA O CLIENTE
