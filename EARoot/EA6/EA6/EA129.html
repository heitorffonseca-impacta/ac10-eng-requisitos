B1 - Agendamento solicitado pelo paciente Basic Path
Details:
<b>Solicitar agendamento pelo paciente</b><br /></p><p><br /></p><p>Este caso de uso tem como objetivo permitir que o paciente solicite um agendamento<br /></p>
Steps:
1. O PACIENTE SELECIONA A OPÇÃO DE AGENDAR CONSULTA
2. O SISTEMA EXIBE A TELA DE AGENDAMENTO
3. O PACIENTE SELECIONA O SERVIÇO DESEJADO E OS PROCEDIMENTOS
4. O SISTEMA EXIBE OS PROFISSIONAIS PARA AQUELES PROCEDIMENTOS
5. O PACIENTE SELECIONA O PROFISSIONAL DESEJADO
6. O SISTEMA EXIBE AS DATAS E HORÁRIOS DISPONÍVEIS PARA PROFISSIONAL
7. O PACIENTE SELECIONA UMA DATA E UM HORÁRIO DISPONÍVEL E CLICA NO BOTÃO “PROSSEGUIR”
8. O SISTEMA EXIBE UMA TELA DE CONFIRMAÇÃO DOS DADOS PREENCHIDOS E O VALOR DE ACORDO COM OS PROCEDIMENTOS SELECIONADOS
9. O PACIENTE CLICA NO BOTÃO SOLICITAR AGENDAMENTO
10. O SISTEMA REGISTRA A SOLICITAÇÃO DE AGENDAMENTO E NOTIFICA O PROFISSIONAL
